# Cafe Ordering System

A simple command-line cafe ordering system built with Python.

## Description
This program simulates a cafe menu where customers can order items, specify quantities, and get a total bill. It displays a menu with prices and handles multiple orders.

## Features
- Displays menu with prices
- Allows ordering multiple items with quantities
- Calculates and displays total bill
- Command-line interface

## Requirements
- Python 3.x

## How to Run
Run the script with Python:
```
python kafe.py
```

## Usage
1. Run the program
2. View the menu
3. Enter item names to order (case-insensitive)
4. Specify quantity
5. Continue ordering or finish
6. View your bill
